    Online Application for UCI Graduate Admissions

    Welcome to the online application for all Master's, Ph.D., and Summer Research programs offered at UCI, except for programs offered through the Business, Law, and Medical Schools.

    • For Graduate Admissions, please refer to the general application instructions and specific program related instructions prior to filling out the application. If you have questions about program-specific evaluative criteria, processes, requirements or deadlines, please contact the program directly.
    • For Summer Research Programs, please refer to the Graduate Preparation Program website for more information prior to filling out the application.

    Incomplete applications will not be reviewed.

      Contact us if you have general questions about the admissions process or experience any technical issues during the application process.
     

    Returning users:

    Log in to continue an application.

    First-time users:

    Create an account to start a new application.
